# BOIDS
BOIDS é uma simulação inventada por Craig W. Reynolds em 1986. Utiliza três regras para reger um movimento semelhante ao de uma revoada de pássaros em 2D. Aqui, tentamos implementar uma abordagem de cardume em Python. Créditos para Álvaro Essinger e Pedro Widniczck pela abordagem original. Acesso em: https://fiscomp.if.ufrgs.br/index.php/BOIDS
